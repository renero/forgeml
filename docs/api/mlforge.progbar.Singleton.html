class Singleton(type):

View In Hierarchy

A metaclass that allows a class to have only one instance. This metaclass ensures that only one instance of a class is created and provides a global point of access to that instance. Usage: class MyClass(metaclass=Singleton): # class definition Note: This metaclass should be used as a metaclass for the class that you want to make a singleton.
